## Descripción
Este proyecto es un sistema desarrollado en Python para gestionar información básica de mascotas, utilizando Programación Orientada a Objetos (POO) y una arquitectura modular para mejorar la organización del código.

## Estructura del proyecto
- `modelos/`: Contiene la clase `Mascota` que define los atributos.
- `servicios/`: Contiene la clase `GestionMascotas` con la lógica para registrar y listar.
- `main.py`: Punto de entrada del sistema.

## Reflexión sobre la Modularidad y POO
La combinación de la Programación Orientada a Objetos con la modularidad es fundamental. La POO nos permite modelar objetos del mundo real (como mascotas) con sus características, mientras que la modularidad nos ayuda a separar las responsabilidades en archivos independientes. Esto hace que el código sea mucho más ordenado, fácil de mantener y permite que varios desarrolladores trabajen en distintas partes del sistema sin conflictos.

## Cómo ejecutarlo
1. Asegúrate de tener Python instalado.
2. Abre la terminal en la carpeta raíz.
3. Ejecuta: `py main.py`
